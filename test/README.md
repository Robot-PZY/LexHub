# LexHub 演示测试用例

三套 **互不重叠** 的演示场景，覆盖劳动争议、公司治理、数据合规，每套含 PDF 材料 + `prompt.md`。

| 文件夹 | 建议场景 | 材料类型 |
|--------|----------|----------|
| `case-01-labor-dispute/` | 争议解决 | 解除通知、工资明细、考勤、仲裁草稿 |
| `case-02-corporate-governance/` | 公司事务 | 章程、股东会决议、股权结构、董事辞任 |
| `case-03-data-compliance/` | 合规知产 | 个保影响评估、投诉、SDK 清单、约谈通知 |

## 使用方式

1. 启动 LexHub，登录用户端
2. 智能工作台选择上表对应场景
3. 上传该文件夹内 **全部 PDF**
4. 将 `prompt.md` 中的任务描述粘贴到工作台后提交

## 重新生成

```bash
python generate_materials.py
```

> 全部为虚构主体与事实，仅供系统联调，不构成法律意见。
