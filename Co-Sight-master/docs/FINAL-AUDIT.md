## 本轮确认并修复

- 模型偶尔省略 `parallel_groups`：现根据 DAG 拓扑层级确定性补全，不再影响并行展示与验收。
- 智能体注册工具与统一工具目录不一致：补齐 `file_find_in_content` 目录项。
- 模板残留人工审阅/复核前置措辞：改为核验智能体自动质量检查。
- 前端入口包约 529KB：路由懒加载后降至约 255KB，消除 500KB 构建警告。
- 启动入口只运行后端且过早打开旧静态页面：改为前后端一键启动、端口复用、健康等待和浏览器自动打开。
- Windows 执行策略会阻止 `npm.ps1`：启动器直接使用 `npm.cmd`。

## 当前非阻塞问题

1. 后端冷启动约 60–100 秒，主要发生在搜索路由、知识库和模型工具导入阶段。一键启动器已给出 10 秒一次的进度提示并等待最多 180 秒；比赛演示应提前启动预热。
2. 真实链路中外部检索或智能体可能触发工具重试预算。这些错误会作为结构化事件展示，任务仍能确定性收敛并输出降级结果；正式演示前建议确认第三方 API 配额和网络。
3. 前端 CSS 仍约 283KB，但已压缩且不阻塞首屏；后续可在视觉方案完全冻结后清理历史样式。
4. 管理端部分低频操作仍使用浏览器提示框，核心工作台、回放和导出已改为内联可恢复反馈。

## 启动方式

- 启动：双击 `start-cosight.bat`

- 仅检查环境：`python start_lexhub.py --check`
- 完整审计：`python scripts/full_chain_audit.py --timeout 180`
