# 律枢 LexHub · 3 分钟演示脚本

> 建议录屏分辨率 1920×1080，旁白简洁，突出 Co-Sight 而非普通法律后台。

## 0:00–0:30 能力总览（/board）

**画面**：打开 `/board`

**旁白要点**：
- 律枢 LexHub，基于 **中兴 Co-Sight** 二次开发的法律业务智能服务平台示范
- 赛道六开放创新：合同审查、争议解决、合规等场景
- 指向「赛题能力映射表」：多智能体、DAG、工具链、多跳、性能对比
- 快速滑过「提交就绪度」与「3 分钟演示路径」

## 0:30–1:30 真实执行（/workspace）

**画面**：进入工作台，输入合同审查类任务，可上传材料

**旁白要点**：
- 不是固定流水线，而是 **Co-Sight DAG 动态编排**
- 展示：DAG 执行视图、多智能体运行证据、工具链覆盖（≥3 类）
- 若有真实 replay：指出工具调用来自 WebSocket 真实记录

## 1:30–2:00 调度网络（/agents）

**画面**：`/agents` 智能体调度页

**旁白要点**：
- 5 个智能体 **按状态亮灭**，非写死顺序
- 材料不足→证据质检；引用不足→法规研究；导出前→强制审查

## 2:00–2:40 回放与导出（/replay + 导出）

**画面**：`/replay` 选一条历史记录；演示导出 DOCX/PDF

**旁白要点**：
- 全链路 **replay.json** 可复核
- 导出内容来自 **真实执行记录**，含执行溯源附录
- 强调「可信、可复核」

## 2:40–3:00 管理与商业（/admin）

**画面**：管理端模型配置 + `/admin/users` 会员标注

**旁白要点**：
- 每智能体可独立配模型与 API Key
- 个人会员三档为商业化规划（体验/Pro/Ultra），机构走定制
- 同一 Co-Sight 底座可复制到多场景

## 收尾金句

> 「律枢 LexHub 用 Co-Sight 把法律任务从人工串行协作，升级为可编排、可工具化、可回放的多智能体协同系统。」
