# 律枢 LexHub 会员与商业化规划（比赛 Demo 版）

> 本文档描述个人会员分层与商业化路径，用于创新总结报告与答辩材料。  
> **本次比赛阶段仅实现会员等级标注与管理端查看，不做功能权限拦截与真实计费。**

---

## 1. 定位说明

律枢 LexHub 面向个人法律从业者提供基于中兴 Co-Sight 的法律业务任务编排服务。商业化采用：

- **个人用户**：体验版 / 专业版 / 旗舰版 三档订阅（规划）
- **机构客户**：定制交付（本次不展开标准团队档）

答辩推荐表述：

> 本次比赛验证 Co-Sight 在法律场景的可行性与分层商业路径；会员体系当前为 **产品规划 + Demo 标注**，非已上线收费系统。

---

## 2. 个人会员三档

| 档位 | 英文 ID | 定位 |
|------|---------|------|
| 体验版 | `trial` | 功能精简，用于体验 Co-Sight 编排 |
| 专业版 | `pro` | 功能全开，标准办案闭环 |
| 旗舰版 | `ultra` | Pro 全开 + 更高用量 + 深度增值（规划） |

### 2.1 体验版（Trial）

- 智能体：部分开放（规划）
- 工具链：基础能力（规划）
- 任务次数：较少（规划）
- 导出：受限或带水印（规划）

### 2.2 专业版（Pro）

- 5 个法律智能体全开
- 完整 DAG 动态编排
- 工具链 ≥3 类
- DOCX / PDF 真实执行导出
- 回放与归档

### 2.3 旗舰版（Ultra）

在 Pro 基础上增加（规划，本次不实现拦截）：

- 更高月任务次数与导出额度
- 高级工具包（企业核验、深度法规 RAG 等）
- 执行溯源增强、专属模板包
- 并行案件与优先调度（概念层）

### 2.4 机构定制

律所、企业法务、政企客户走 **定制方案**（私有化、行业工作流、专属工具对接），不纳入个人三档菜单。

---

## 3. 收费逻辑（规划，非最终实现）

```
个人：订阅制（月/年） + Ultra 可选超量包
机构：定制报价（项目制 / 年费）
```

价格仅作商业论证量级，不作为比赛交付承诺：

- 体验版：免费
- 专业版：百元～数百元/月量级（规划）
- 旗舰版：约为 Pro 的 1.5～2.5 倍（规划）
- 定制：面议

## 4. 本次比赛已实现（Demo）

### 4.1 用户端

- 登录/注册后，顶部显示会员标签（体验版 / 专业版 / 旗舰版）
- 新注册用户默认 **体验版**
- **不按会员等级限制功能**（避免未完成权限体系被评委追问）

### 4.2 管理端

- 路由：`/admin/users`
- 能力：
  - 查看用户列表（姓名、账号、机构、会员等级、状态、最近活跃）
  - 修改用户会员等级（本地 Demo 存储）
  - 会员分布统计（体验 / Pro / Ultra 数量）

### 4.3 数据存储

- 浏览器 `localStorage` 键：`lexhub:membershipUsers`
- 当前登录用户：`lexhub:demoUser` 含 `membershipTier` 字段
- 预置 Demo 用户见 `cosight_frontend/src/mocks/membership-users.ts`

## 5. 可复制性论证（赛题相关）

同一套 Co-Sight 底座可通过以下方式复制到不同客户：

1. **工作流配置**：`config/legal-workflow.json` 按场景切换
2. **工具链接入**：管理端 API 配置按客户启用
3. **知识库模板**：按行业打包
4. **会员档位**：按客户规模切换能力边界与用量（规划）
